## ამ ეტაპის ფუნქციონალი
- **მენიუ** — Navigation Drawer (Inventory, About).
- **ლისთი** — LazyColumn ავტომობილების სიით.
- **CRUD** — მანქანის დამატება, რედაქტირება, წაშლა ვალიდაციით.
- **MVVM** — ViewModel + Repository + Room.
- **ბაზა** — Room (ლოკალური SQLite).
